## 🧮 מתודולוגיה אקטוארית

### נוסחת החישוב

$$DBO = \sum_{t=1}^{T} \; p_t \;\cdot\; B_t \;\cdot\; v_t$$

כאשר:
| סימון | משמעות |
|-------|---------|
| $t$ | שנה עתידית |
| $p_t$ | הסתברות מצטברת להישרדות עד שנה $t$ |
| $B_t$ | תשלום הטבה צפוי באירוע בשנה $t$ |
| $v_t$ | גורם היוון לשנה $t$ |

### הסתברויות יציאה

$$p_t = \prod_{s=1}^{t-1}(1 - q_s^{dismissal} - q_s^{resignation} - q_s^{mortality})$$

- **פיטורין** — שיעורים לפי רצועות גיל מגליון ההנחות  
- **התפטרות** — שיעורים לפי רצועות גיל מגליון ההנחות  
- **תמותה** — מלוחות התמותה הרשמיים (גברים/נשים)

### הטבות לפי אירוע

| אירוע | הטבה (ותק < 5) | הטבה (ותק ≥ 5) |
|-------|----------------|----------------|
| פיטורין | שכר × ותק × מקדם סעיף 14 | `max`(פיצויים חוקיים, נכסי קרן) |
| התפטרות | נכסי קרן | נכסי קרן |
| תמותה | שכר × ותק × מקדם סעיף 14 | `max`(פיצויים חוקיים, נכסי קרן) |
| פרישה | שכר × ותק × מקדם סעיף 14 | `max`(פיצויים חוקיים, נכסי קרן) |

### עקום היוון
שיעור ההיוון נלקח מעקום ריביות שנתי המופיע בגליון ההנחות — שיעור שונה לכל שנת תחזית.
